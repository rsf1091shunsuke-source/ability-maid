# 🃏 ババ抜き - リアルタイム対戦

2人がそれぞれのブラウザから対戦できる1対1の新しいババ抜きゲームです。

## 遊び方

1. [https://ability-maid.vercel.app/](https://ability-maid.vercel.app/) を開く
2. 名前を入力して「新しい部屋を作る」をクリック
3. 表示されたルームコードを相手に共有する
4. 相手が同じURLを開いてルームコードを入力して参加
5. 対戦開始！

## 使用技術

- **Next.js 14** - アプリの構造・画面の動きを管理
- **Firebase Firestore** - リアルタイムでゲーム状態を同期
- **Vercel** - デプロイ

## 工夫した点

リアルタイム同期にはFirestoreの `onSnapshot` を使用しました。
片方のプレイヤーがカードを引くと、Firestoreのデータが更新され、もう片方のブラウザが自動で反応して画面が切り替わります。これはLivePoker Assistのブラインドタイマーの同期で使っていた経験があったので応用しました。

ここからはババ抜きのルールに関してなのであまり関係はないかもしれないのですが2人でするババ抜きということで最初は普通のルールのババ抜きを作りました。しかし、

1.どちらにジョーカーがあるかがどちらのプレイヤーもわかる

2.ジョーカー以外を取ると絶対にそろうのでジョーカー以外をとるゲームになってしまっている。

3.現実でもそのルールでババ抜きができるため2人でババ抜きをしたい人で離れている人だけがターゲットとなる

という問題がありました。そこでアビリティババ抜きというものを考えました。
簡単な説明としては能力が各カードについており同じカードがそろったら能力が使うことができるというルールです。戦略的かつ運の要素もあるので上の1~3の問題を解決することができたと思います。

## アビリティババ抜きのルール

### 基本ルール
- 2人でプレイするリアルタイムのババ抜きです
- 1.最後にジョーカーを持っているプレイヤーの負けです
- 2.手札が0枚になったら勝ちです
- 1.2の両方ではなく2つのゲームの勝ち方があるといくことです

### ゲームの流れ
1. 各プレイヤーに全て異なる能力の手札が7枚配られます
2. 自分のターンになると山札から自動でカードを引きます
3. 次に相手の手札からカードを1枚引きます
4. 同じ能力カードが2枚揃うと能力が使えるようになります
5. 5秒のカウントダウン中に能力を発動できます
6. ターン終了後に相手のターンへ移ります

### カードの種類

#### 戦略系（4枚ずつ）
| 能力 | 効果 |
|------|------|
| 覗き見 | 相手の手札を5秒間見る |
| マーカー | 相手の手札1枚の場所を把握できる |
| 封じ込め | 次のターン相手の手札がシャッフルされない |
| 情報封鎖 | 相手の覗き見・マーカーを無効化する |
| 反射 | 相手の次の能力を跳ね返す |
| 無効 | 相手の次の能力を無効化する |
| 返却 | 手札1枚を山札に戻す |
| 交換 | 手札を全部入れ替える※ |
| 手渡し | 自分の手札から1枚選んで相手に渡す |
| 囮 | 次に相手が引くとき1枚指定して引かせない※ |

※条件付き能力：交換は相手の手札が自分より3枚以上多いとき、囮は相手の手札が3枚以下のときのみ発動可能

#### 運系（6枚ずつ）
| 能力 | 効果 |
|------|------|
| 山引き | 山札からランダムに2枚引く |
| 全公開 | 相手の手札を10秒間全て表向きにする |

#### 呪い系（4枚ずつ）
| 能力 | 効果 |
|------|------|
| 暴露 | ペアが揃った瞬間、自分の手札が5秒間相手に公開される |

#### ジョーカー（1枚）
引いた次のターン開始時に強力な効果が発動します。
最後まで持っていたプレイヤーの負けです。
